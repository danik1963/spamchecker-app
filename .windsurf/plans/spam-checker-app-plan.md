# SpamChecker - План мобильного приложения

Кроссплатформенное мобильное приложение (iOS/Android) для ведения публичной базы данных спам-номеров и мошенников с возможностью добавления комментариев.

---

## 🎯 Основной функционал

| Функция | Описание |
|---------|----------|
| **Поиск номера** | Быстрый поиск по базе спам-номеров |
| **Добавление номера** | Пользователи добавляют номера с категорией (спам/мошенник) |
| **Комментарии** | Обсуждение и отзывы под каждым номером |
| **Рейтинг опасности** | Автоматический рейтинг на основе жалоб |
| **История просмотров** | Локальная история проверенных номеров |

---

## 📱 Экраны приложения

### 1. Главный экран (Home)
- Поисковая строка для проверки номера
- Последние добавленные спам-номера
- Статистика базы данных

### 2. Результат поиска (Search Result)
- Информация о номере
- Категория: Спам / Мошенник / Неизвестен
- Количество жалоб
- Список комментариев

### 3. Добавление номера (Add Number)
- Ввод номера телефона
- Выбор категории (спам/мошенник)
- Описание ситуации
- Кнопка отправки

### 4. Детали номера (Number Details)
- Полная информация о номере
- Все комментарии с датами
- Форма добавления комментария
- Кнопка "Пожаловаться"

### 5. Профиль/Настройки (Profile)
- Статистика добавленных номеров
- История проверок
- Настройки уведомлений

## 🔄 API Endpoints

```
GET    /api/phones/:number     - Получить информацию о номере
POST   /api/phones             - Добавить новый номер
GET    /api/phones/:id/comments - Получить комментарии
POST   /api/phones/:id/comments - Добавить комментарий
POST   /api/phones/:id/report  - Пожаловаться на номер
GET    /api/phones/recent      - Последние добавленные номера
POST   /api/auth/register      - Регистрация
POST   /api/auth/login         - Вход
```

## 🛡 Защита от злоупотреблений

```
Механизм защиты:
├── Порог жалоб: номер = "спам" только после 3+ уникальных жалоб
├── Device fingerprint: 1 устройство = 1 голос за номер
├── IP лимит: макс 10 добавлений с одного IP в день
└── Статусы номера: "На проверке" (1-2 жалобы) → "Спам" (3+ жалоб)
```

### Логика отображения:
- **0-2 жалобы** → Номер в статусе "На проверке", показывается с предупреждением
- **3+ жалобы** → Номер подтверждён как спам/мошенник
